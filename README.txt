Deskripsi Program :
Program ini mengenai "Sistem Informasi Apotek" dimana terdapat beberapa fitur terdiri dari
login, menu dan beberapa fitur lain.

+ Pada menu login terdapat fitur untuk mengisi username dan password serta tombol login
  - username yang diisi dapat digunakan "doaibu"
  - password yang diisi dapat menggunakan "doaibu"
  - hakakses yang diisi dapat menggunakan administrator

+ Pada menuutama terdapat fitur :
  - Background
  - Keluar
  - data barang
  - transaksi
  - laporan

+ Clone github
  - git clone https://github.com/ndiasofia/Projek-Akhir-UAS-PBO-.git
+ Run program
  - project dapat dibuka menggunakan netbeans untuk memudahkan running, lalu menuju ke
    folder gui dan run file login, untuk memulai eksekusi program
